# AI-Language-Assistant
구글 번역기와 Deep Translator API를 활용한 맞춤형 AI 번역 프로그램입니다.

## 프로그램 주요 기능
- 여러개의 언어 선택 가능: 다국어 번역 지원
- 자연스러운 어조 선택: API를 통한 자연스러운 번역
- 간단한 방식으로 번역 가능: 쉽고 간편한 UI

## 기술
- 구글 번역기 API
- DeepL API
