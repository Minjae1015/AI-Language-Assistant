# AI-Language-Assistant
구글 번역기와 Deep Translator API를 활용한 맞춤형 AI 번역 프로그램입니다.

## 프로그램 주요 기능
- 입력 언어를 감지해서 한국어로 번역
- 자연스러운 어조 선택

## 기술
- 구글 번역기 API
- DeepL API
